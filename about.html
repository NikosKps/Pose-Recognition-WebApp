Built for accuracy. Engineered for privacy.

What it is

A web application that uses machine learning to analyze and classify human body posture in real time, entirely within your browser — no uploads, no servers, no installation.

Pose detection runs on TensorFlow.js. A custom-trained classifier maps the resulting keypoints to one of seven posture classes, frame by frame.

RuntimeBrowser / WebGL
BackbonePoseNet (tfjs)
ClassifierTeachable Machine
Latency≤ 50ms / frame

How it works

The app is powered by Google's Teachable Machine, running a custom-trained pose classification model directly in your browser. It detects body keypoints from the camera feed and classifies the result across 7 trained classes.

All processing happens locally. Your camera stream never leaves the device — there is no backend to leak it to.

Classes7 postures
Test length15 seconds
Confirm window12 frames
Min confidence0.45
